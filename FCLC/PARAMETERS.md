## § Differential Privacy

| Параметр | Значение | Описание |
|----------|----------|----------|
| `epsilon` | ≤ 1.0 | Бюджет DP (строгий) |
| `delta` | 1e-5 | Вероятность нарушения DP |
| Механизм | Gaussian | Шум для градиентов |
| Клиппинг градиентов | L2 norm ≤ 1.0 | До добавления шума |
