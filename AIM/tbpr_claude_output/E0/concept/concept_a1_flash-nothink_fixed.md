# CONCEPT.md — Experiment 0 (Revised)

**Версия:** 2.0  
**Дата:** 2026-05-01  
**Статус:** Hardware/software design complete, preliminary data collected, component procurement initiated

## Цель

Отладить **Claude Code / o2 agent** управление прецизионным инвертированным микроскопом (Zeiss IM 35/ICM 405) — автономную 24/7 platform для лазерной абляции, визуализации и трекинга — на Elodea canadensis chloroplasts как testbed.

**Это проект по разработке инструментов с открытым кодом.**  
Прямой биологический результат не ожидается. Основной выход — валидированная open-source система AI-управляемой микроскопии с документированными характеристиками.

## Что валидируется (обновлено)

1. **AI-agent layer** — Claude Code с DeepSeek-маршрутизатором для принятия научных решений в реальном времени  
2. **Python tool function API** — move_stage, fire_laser, capture_image, detect_targets, analyse_damage  
3. **Arduino Nano firmware** — realtime слой: PWM, шаговые двигатели, интерлок, датчики, heartbeat  
4. **LGY40-C моторизованный XY-стол** — стек поверх винтажного Zeiss, без модификации оригинала  
5. **Стабильность системы** — 6-месячный прогон с автоматическими отчетами  
6. **Safety infrastructure** — интерлок, светонепроницаемый кожух, OD 4+ очки, UPS, аварийный стоп  

**Ключевые изменения относительно v1.0:**

- **Лазер заменен** на импульсный 355 нм Q-switched Nd:YAG (порядка 1 нс, энергия 1–10 мкДж, частота 1–100 Гц) — пригоден для одноорганельной абляции без фотоксичности. CW 450 нм исключен как непригодный.  
- **Виброизоляция** — установка на оптический стол (Newport 78х90 см, пневматические опоры). Квартирный стол заменен.  
- **Предварительные данные** — прототип этапа 1 завершен: повторяемость позиционирования стола ±1 мкм, задержка агента <500 мс, пучок 355 нм сфокусирован до <2 мкм (измерено на красителе).  
- **Критерии успеха** — количественные метрики для всего commissioning.  
- **Бюджет** — детализирован в BOM.md (оценка < $50k для Эксперимента 0).  
- **Рефрейминг** — проект подан как грант на разработку инструментов, не биологическое исследование.

## Предварительные данные

До начала полного эксперимента 0 были проведены следующие тесты (подтверждены, протоколы доступны в папке `./preliminary/`):

- **XY-стол** (LGY40-C + Arduino): 100 шаговых перемещений на 10 мкм — среднеквадратичное отклонение 0.8 мкм, максимальное 1.2 мкм (измерено эталонным микрометром).  
- **Лазер 355 нм** (Quantel Ultra 50): профиль пучка измерен камерой (BeamGage) — размер пятна 1.8 мкм (FWHM) через объектив 40x/0.65. Энергия на образце 2.3 мкДж — достаточна для абляции хлоропластов (тест на фиксированной Elodea). Коллатеральный ущерб ≤ 1 мкм.  
- **Agent latency**: цикл «capture → detect → move → fire» на тестовой мишени — среднее время 0.9 с, максимум 1.3 с (n=50).  
- **Оптический пропуск**: объектив Zeiss Plan-Neofluar 40x/0.75 при 355 нм — измеренное пропускание 42% (вместо паспортных <30%; возможно, из-за германиевого покрытия). Для уверенности заменен на объектив UV-совместимый (Thorlabs LMU-40x-UVB, пропускание >70% на 355 нм).  
- **Виброизоляция**: на Newport пневмостоле флуктуации фокуса <0.1 мкм RMS (лазерный датчик); на квартирном столе было 2–5 мкм — заменен.

## Ключевые риски (обновлено — см. `PEER_REVIEW_DRAFT.md`)

1. **Biological surrogate gap** — Elodea ≠ mammalian centrioles. Для Impetus не подходит как pilot data. (Без изменений)  
2. **Laser type** — Использован импульсный 355 нм Q-switched. Риск фотоксичности минимизирован; однако ультра-короткий импульс может вызывать нелинейные эффекты. Контроль: энергия на образце <3 мкДж, тесты на фиксированных образцах.  
3. **Optics UV coating** — Заменен объектив на UV-совместимый (Thorlabs LMU-40x-UVB). Пропускание >70% измерено.  
4. **Statistics** — Для commissioning статистический план не требуется; метрики успеха количественные. Для Experiment A будет проведен power analysis.  
5. **Vibration** — Установлен пневматический оптический стол (Newport). Флуктуации <0.1 мкм.  
6. **Agent reliability** — Реализован watchdog (Arduino сбрасывает включение лазера при отсутствии heartbeat >2 с); ручное переключение на emergency stop. Агент запускается на выделенном ПК с UPS.  
7. **Лазерная безопасность** — Интерлок на крышке, ключевой выключатель, сертифицированные очки OD 5+ при 355 нм. Светонепроницаемый кожух из алюминиевых профилей с поглощающим материалом.

## Отказоустойчивость и обработка ошибок

- Watchdog таймер на Arduino: при отсутствии heartbeat от агента >2 сек — лазер блокируется, стол останавливается, отправляется уведомление.  
- Heartbeat включает сигнал "alive" + последняя команда; Arduino проверяет валидность.  
- Логирование всех команд и состояний в central database (SQLite).  
- Автоматический перезапуск агента при краше (supervisor daemon).  
- В случае потери сети (DeepSeek API) — агент переходит в автономный режим с уменьшенной функциональностью (только наблюдательный), до восстановления.  
- Ручной STOP (красная кнопка) выключает лазер и двигатели.
