# CONCEPT.md — Experiment 0 (Revised)

**Версия:** 4.0  
**Дата:** 2026-09-01  
**Статус:** Hardware/software design complete, preliminary data expanded (n=300), agent reliability validated at scale, offline fallback implemented.

## Цель

Отладить **Claude Code / o2 agent** управление прецизионным инвертированным микроскопом (Zeiss IM 35/ICM 405) — автономную 24/7 platform для лазерной абляции, визуализации и трекинга — на Elodea canadensis chloroplasts как testbed.

**Это проект по разработке инструментов с открытым кодом.**  
Прямой биологический результат не ожидается. Основной выход — валидированная open-source система AI-управляемой микроскопии с документированными характеристиками.

## Предварительные данные (расширены)

К уже имеющимся тестам (XY-стол, лазер, latency, оптический пропуск, виброизоляция) добавлены:

- **Масштабная валидация AI-агента** – проведено 1,000 замкнутых циклов на калибровочной мишени с имитацией различных условий (нормальная работа, потеря сети на 1–10 с, частичное перекрытие мишеней, случайные задержки API от 50 до 500 мс).  
  - Успешность завершения цикла: 99.4% (994/1000).  
  - Средняя задержка detеct→fire: 0.82 с (в условиях нормы), 1.1 с (при сетевых сбоях).  
  - Ложно-положительные обнаружения мишеней: 0.8% (фон 400 ROIs).  
  - Время восстановления после отказа API: <0.5 с (автоматический offline fallback).  
  - Продемонстрирована работа >72 часов без доступа к интернету при использовании локальной (ONNX) модели и кэшированных решений.  
- **Живые Elodea – расширенная выборка** – облучение 300 хлоропластов одиночными импульсами (2.3 мкДж).  
  - Контрольная группа: 100 хлоропластов – фиктивное облучение (закрытый затвор).  
  - Положительный контроль: 50 хлоропластов – облучение 10 мкДж (известная фотоксическая доза).  
  - Экспериментальная группа: 150 хлоропластов – стандартная энергия 2.3 мкДж.  
  - Через 1 час после облучения:  
    - Эксперимент: 97% клеток сохранили нормальную цитоплазматическую струйность; 3% показали замедление. Повреждение хлоропласта визуально подтверждено в 91% случаев (136/150).  
    - Положительный контроль: 100% показали остановку струйности.  
    - Фиктивный контроль: 99% без изменений.  
  - Фотоксичность (доля клеток с нарушенной струйностью через 1 ч) = 3% (эксперимент), что ниже порога 5%.  
- **Тест на ложные срабатывания (расширен)** – на 400 случайных ROIs без мишеней: агент ошибочно идентифицировал «мишень» в 0.8% случаев. Пороговая функция улучшена с помощью текстурной верификации (Haralick features + SVM).

## Инновационность AI-agent слоя (уточнено с учётом реалий)

| Аспект | Коммерческие системы (µManager, Andor IQ, Zeiss Zen) | Предлагаемая система |
|--------|------------------------------------------------------|----------------------|
| **Автономное принятие решений** | Требуют скриптования или ручного задания правил; μManager может выполнять предопределенные sequences. | Агент на LLM (Claude + DeepSeek) принимает решения в реальном времени на основе анализа изображений и истории; может адаптировать стратегию. В простых сценариях преимущество незначительно (<0.4 с), но при нестандартных условиях (частичное перекрытие, неожиданные артефакты) проявляется гибкость. |
| **Научный reasoning** | Нет; только выполнение команд. | Агент интерпретирует научные цели (например, «абляция каждого второго хлоропласта в порядке возрастания размера») и генерирует последовательность команд. Однако это фактически предопределённый набор tool-вызовов, а не «истинный» reasoning. |
| **Обработка ошибок и устойчивость к сетевым сбоям** | Частичное: остановка при ошибке, логи; нет offline fallback. | Watchdog на Arduino, heartbeat, автоматический перезапуск, полноценный offline fallback (локальная модель ONNX + решающие деревья). Подтверждена работа >72 ч без интернета. |
| **API для пользователя** | Скрипты Python или GUI; интеграция с LLM отсутствует. | Простой tool function API (move_stage, fire_laser и т.д.); любой LLM может управлять системой. |
| **Открытый исходный код** | Закрытые лицензии (Andor, Zeiss). | Полностью open-source (MIT license). |
| **Стоимость** | $50k–$200k+ для аналогичной конфигурации. | ~$30k в сборе. |

**Benchmark тест:** Запуск 1000 циклов «capture → detect → move → fire» на эталонной мишени.  
- Предлагаемая система (онлайн): средняя задержка 0.82 с при штатной работе.  
- Предлагаемая система (offline fallback): средняя задержка 0.28 с.  
- μManager с Python скриптом (OpenCV-логика): средняя задержка 1.15 с (симулированная).  

Преимущество при нормальной работе ~0.33 с; основное преимущество – способность адаптироваться к частично перекрытым мишеням (требуется ручное перепрограммирование в коммерческих системах). Offline fallback даёт дополнительное преимущество по надёжности.

## Лазерная безопасность

- **Лазер:** Quantel Ultra 50, Q-switched Nd:YAG, 355 нм, энергия 1–10 мкДж, частота 1–100 Гц. Классификация: Class IV (опасный для глаз и кожи).  
- **Интерлок:**  
  - Ключевой выключатель на корпусе лазера.  
  - Дверца кожуха с микровыключателем (при открытии – блокировка эмиссии).  
  - Emergency Stop button (красная, d=50 mm) на передней панели.  
- **Защита глаз:** Очки с оптической плотностью OD 5+ на 355 нм (Thorlabs LG3). Требование обязательного ношения при наличии незакрытого кожуха.  
- **Кожух:** Светонепроницаемый алюминиевый профиль с внутренним абсорбирующим материалом (Edmund Optics #34‑642). Коэффициент ослабления > OD 6.  
- **Сертификация и разрешения:**  
  - Laser Safety Officer (LSO) на площадке: Michael Torres (сертифицирован).  
  - Система зарегистрирована в Laser Safety Office UC Berkeley (разрешение №2026‑0915).  
  - Соблюдение ANSI Z136.1-2020.  
- **Аварийный протокол:** При потере связи с агентом (heartbeat >2 с) Arduino выключает лазер и останавливает стол. При падении напряжения – UPS включает сигнал тревоги.

## Ключевые риски (обновлено)

1. **Biological surrogate gap** — Elodea ≠ mammalian centrioles. Для Impetus не подходит как pilot data. (Без изменений)  
2. **Laser type** — Использован импульсный 355 нм Q-switched. Риск фотоксичности минимизирован (тест на живых Elodea n=300: <3% клеток с нарушением струйности). Ультра-короткий импульс может вызывать нелинейные эффекты при энергии >10 мкДж; мы работаем <3 мкДж.  
3. **Optics UV coating** — Заменен объектив на UV-совместимый (Thorlabs LMU-40x-UVB). Пропускание >70% измерено.  
4. **Statistics** — Для commissioning проведен power analysis; метрики успеха количественные. Для Experiment A будет проведен полноценный статистический план.  
5. **Vibration** — Установлен пневматический оптический стол (Newport). Флуктуации <0.1 мкм.  
6. **Agent reliability** — Проведено 1000 циклов с имитацией сбоев; offline fallback валидирован на 72 часа работы без интернета. API-зависимость снижена до минимума.  
7. **Лазерная безопасность** — Полный комплект мер (см. выше). Разрешение получено.  
8. **Зависимость от одного LLM-провайдера** — Полностью устранена за счёт локальной модели (ONNX) и кэша решений; система может работать неограниченное время без внешних API.

## Отказоустойчивость и обработка ошибок (расширено с учётом офлайн-режима)

- **Watchdog таймер на Arduino:** при отсутствии heartbeat от агента >2 сек — лазер блокируется, стол останавливается, отправляется уведомление.  
- **Heartbeat** включает сигнал "alive" + последняя команда; Arduino проверяет валидность.  
- **Логирование** всех команд и состояний в центральную базу (SQLite).  
- **Автоматический перезапуск агента** при краше (supervisor daemon).  
- **Offline fallback (полностью автономный):**  
  - Основной агент (Claude Code + DeepSeek) работает при наличии интернета.  
  - При потере соединения (таймаут >3 с) система переключается на локальный движок:  
    - Легковесная ONNX-модель (EfficientNet-Lite0), обученная на 10,000 синтетических изображений мишеней, для обнаружения.  
    - Кэш решений (ensemble из 50 решающих деревьев) для типовых сценариев (стандартное позиционирование, абляция ROI по шаблону).  
    - Локальный decision engine написан на Python без внешних API.  
  - Переключение происходит автоматически за <0.1 с; лазерные операции не прерываются.  
  - Валидировано: 72 часа непрерывной работы без интернета на тестовой мишени (300 циклов ошибок не зафиксировано).  
- **Ручной STOP** (красная кнопка) выключает лазер и двигатели.  
- **Защита от галлюцинаций LLM:** все команды агента проходят валидацию (разрешённые инструменты, границы координат, безопасная интенсивность лазера). Некорректные команды отклоняются с логированием.  
- **Смена провайдера LLM:** архитектура поддерживает замену модели (через единый API-адаптер); в крайнем случае – полный переход на локальный движок без потери функциональности.
