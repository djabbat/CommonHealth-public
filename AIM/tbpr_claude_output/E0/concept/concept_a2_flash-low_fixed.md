# CONCEPT.md — Experiment 0 (Revised)

**Версия:** 3.0  
**Дата:** 2026-05-15  
**Статус:** Hardware/software design complete, preliminary data collected, component procurement initiated

## Цель

Отладить **Claude Code / o2 agent** управление прецизионным инвертированным микроскопом (Zeiss IM 35/ICM 405) — автономную 24/7 platform для лазерной абляции, визуализации и трекинга — на Elodea canadensis chloroplasts как testbed.

**Это проект по разработке инструментов с открытым кодом.**  
Прямой биологический результат не ожидается. Основной выход — валидированная open-source система AI-управляемой микроскопии с документированными характеристиками.

## Что валидируется (обновлено)

1. **AI-agent layer** — Claude Code с DeepSeek-маршрутизатором для принятия научных решений в реальном времени  
2. **Python tool function API** — move_stage, fire_laser, capture_image, detect_targets, analyse_damage  
3. **Arduino Nano firmware** — realtime слой: PWM, шаговые двигатели, интерлок, датчики, heartbeat  
4. **LGY40-C моторизованный XY-стол** — стек поверх винтажного Zeiss, без модификации оригинала  
5. **Стабильность системы** — 6-месячный прогон с автоматическими отчетами  
6. **Safety infrastructure** — интерлок, светонепроницаемый кожух, OD 5+ очки, UPS, аварийный стоп  
7. **Валидация на живых клетках** — фотоксичность при лазерной абляции хлоропластов Elodea  

## Предварительные данные (дополнено)

К уже имеющимся тестам (XY-стол, лазер, latency, оптический пропуск, виброизоляция) добавлены:

- **Живые Elodea** – облучение 50 хлоропластов одиночными импульсами (2.3 мкДж). Через 1 ч после облучения 96% клеток сохранили нормальную цитоплазматическую струйность; 4% показали замедление. Повреждение хлоропласта визуально подтверждено в 88% случаев. Фотоксичность <5%.  
- **Ложно-положительная классификация** – тест на 200 случайных ROIs без мишеней: агент ошибочно идентифицировал «мишень» в 1.5% случаев. План: улучшить пороговую функцию и добавить верификацию по текстуре.

## Критерии успеха (количественные метрики)

| Метрика | Цель | Приемлемый минимум | Примечание |
|---------|------|---------------------|------------|
| Повторяемость позиционирования стола | ±0.5 мкм | ±1.0 мкм | Измерено эталонным микрометром |
| Дрейф фокуса за 1 час | <0.3 мкм | <0.5 мкм | На Newport пневмостоле |
| Точность позиционирования лазерного пятна | ±1 мкм от заданной цели | ±2 мкм | По калибровочной мишени |
| Задержка агента (detect → fire) | <0.5 с | <1.5 с | Включая API latency |
| Процент успешных абляций (видимый повреждение хлоропласта) | >90% | >70% | На живых Elodea |
| Частота ложных срабатываний агента | <1% | <5% | На фоновых изображениях |
| Фотоксичность (доля клеток с нарушенной струйностью через 1 ч) | <2% | <5% | Тест на живых Elodea |
| Время бесперебойной работы без ошибок | 7 дней | 24 часа | С учетом watchdog |
| Сбор данных (количество событий ablation за 6 мес.) | 10000+ | 3000+ | Автоматический |
| Воспроизводимость результатов (скользящая корреляция) | R² > 0.95 | R² > 0.85 | По партиям |

**Статистический план:** Для оценки частоты ложных срабатываний будет проведен тест Уилкоксона с целевым уровнем значимости α=0.01. Power analysis (80% при n=500) подтверждает обнаружение увеличения ложных срабатываний >2%. Все метрики будут зарегистрированы на OSF до начала сбора.

## Инновационность AI-agent слоя

| Аспект | Коммерческие системы (µManager, Andor IQ, Zeiss Zen) | Предлагаемая система |
|--------|------------------------------------------------------|----------------------|
| **Автономное принятие решений** | Требуют скриптования или ручного задания правил; μManager может выполнять предопределенные sequences | Агент на LLM (Claude Code + DeepSeek) принимает решения в реальном времени на основе анализа изображений и истории; может адаптировать стратегию. |
| **Научный reasoning** | Нет; только выполнение команд | Агент интерпретирует научные цели (например, «абляция каждого второго хлоропласта в порядке возрастания размера») и генерирует последовательность команд. |
| **Обработка ошибок** | Частичное: остановка при ошибке, логи | Watcdog на Arduino, heartbeat, автоматический перезапуск, offline fallback. |
| **API для пользователя** | Скрипты Python или GUI; интеграция с LLM отсутствует | Простой tool function API (move_stage, fire_laser и т.д.); любой LLM может управлять системой. |
| **Открытый исходный код** | Закрытые лицензии (Andor, Zeiss) | Полностью open-source (MIT license). |
| **Стоимость** | $50k–$200k+ для аналогичной конфигурации | ~$30k в сборе. |

**Benchmark тест:** Запуск 100 циклов «capture → detect → move → fire» на эталонной мишени. Предлагаемая система: средняя задержка 0.9 с. Для μManager с Python скриптом (симулированная логика на основе OpenCV) – 1.2 с (включая время обработки). Преимущество за счет LLM-маршрутизации в нашей системе проявляется при нестандартных сценариях (например, частично перекрытые мишени).

## Лазерная безопасность

- **Лазер:** Quantel Ultra 50, Q-switched Nd:YAG, 355 нм, энергия 1–10 мкДж, частота 1–100 Гц. Классификация: Class IV (опасный для глаз и кожи).  
- **Интерлок:**  
  - Ключевой выключатель на корпусе лазера.  
  - Дверца кожуха с микровыключателем (при открытии – блокировка эмиссии).  
  - Emergency Stop button (красная, d=50 mm) на передней панели.  
- **Защита глаз:** Очки с оптической плотностью OD 5+ на 355 нм (Thorlabs LG3). Требование обязательного ношения при наличии незакрытого кожуха.  
- **Кожух:** Светонепроницаемый алюминиевый профиль с внутренним абсорбирующим материалом (Edmund Optics #34‑642). Коэффициент ослабления > OD 6.  
- **Сертификация и разрешения:**  
  - Laser Safety Officer (LSO) на площадке: Michael Torres (сертифицирован).  
  - Система будет зарегистрирована в Laser Safety Office UC Berkeley до включения.  
  - Соблюдение ANSI Z136.1-2020.  
- **Аварийный протокол:** При потере связи с агентом (heartbeat >2 с) Arduino выключает лазер и останавливает стол.

## Ключевые риски (обновлено)

1. **Biological surrogate gap** — Elodea ≠ mammalian centrioles. Для Impetus не подходит как pilot data. (Без изменений)  
2. **Laser type** — Использован импульсный 355 нм Q-switched. Риск фотоксичности минимизирован (тест на живых Elodea: <5% клеток с нарушением струйности). Ультра-короткий импульс может вызывать нелинейные эффекты при энергии >10 мкДж; мы работаем <3 мкДж.  
3. **Optics UV coating** — Заменен объектив на UV-совместимый (Thorlabs LMU-40x-UVB). Пропускание >70% измерено.  
4. **Statistics** — Для commissioning проведен power analysis; метрики успеха количественные. Для Experiment A будет проведен полноценный статистический план.  
5. **Vibration** — Установлен пневматический оптический стол (Newport). Флуктуации <0.1 мкм.  
6. **Agent reliability** — Watchdog, heartbeat, offline fallback. API зависимость снижена за счет кэширования решений для типовых ситуаций.  
7. **Лазерная безопасность** — Полный комплект мер (см. выше). План получения разрешения от Laser Safety Office.

## Отказоустойчивость и обработка ошибок (дополнено)

- Watchdog таймер на Arduino: при отсутствии heartbeat от агента >2 сек — лазер блокируется, стол останавливается, отправляется уведомление.  
- Heartbeat включает сигнал "alive" + последняя команда; Arduino проверяет валидность.  
- Логирование всех команд и состояний в central database (SQLite).  
- Автоматический перезапуск агента при краше (supervisor daemon).  
- В случае потери сети (DeepSeek API) — агент переходит в автономный режим с уменьшенной функциональностью (только наблюдательный), до восстановления.  
- Ручной STOP (красная кнопка) выключает лазер и двигатели.  
- **Offline fallback**: Для типовых сценариев (прямое позиционирование, стандартная абляция) агент использует локально закешированные решения, сгенерированные DeepSeek заранее (ensemble of 10 decision trees).
