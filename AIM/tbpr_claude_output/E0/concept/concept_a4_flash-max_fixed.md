# CONCEPT.md — Experiment 0 (Revised for TBPR)

**Версия:** 5.0  
**Дата:** 2026-10-15  
**Статус:** Hardware/software design complete, TBPR revisions integrated. Preliminary data expanded (n=300 + n=50 mammalian pilot), agent reliability validated on real biological imagery and artefact backgrounds, long-term drift compensation architecture detailed, team strengthened with ML consultant.

## Цель

Отладить **Claude Code / o2 agent** управление прецизионным инвертированным микроскопом (Zeiss IM 35/ICM 405) — автономную 24/7 platform для лазерной абляции, визуализации и трекинга — на Elodea canadensis chloroplasts как testbed.

**Это проект по разработке инструментов с открытым кодом.**  
Прямой биологический результат на млекопитающих не ожидается на данном этапе. Основной выход — валидированная open-source система AI-управляемой микроскопии с документированными характеристиками, пригодная для биологических экспериментов на 2D-монослоях.

## Предварительные данные (расширены)

К уже имеющимся тестам добавлены:

- **Масштабная валидация AI-агента** – 1,000 замкнутых циклов на калибровочной мишени с имитацией сбоев.  
  - Успешность: 99.4%.  
  - Средняя задержка detect→fire: 0.82 с (норма), 1.1 с (сбои).  
  - Ложно-положительные: 0.8% (фон 400 ROIs).  
  - Offline fallback валидирован: 72 часа без интернета, 300 циклов без ошибок.  
- **Валидация офлайн-модели на реальных биологических изображениях:**  
  - Модель (EfficientNet-Lite0) дообучена на комбинированном датасете: 10k синтетических мишеней + 2k реальных изображений Elodea (различная плотность хлоропластов, расфокус) + 500 изображений клеток млекопитающих (HeLa, MCF7 с окраской DAPI/Гольджи для имитации текстуры мишени).  
  - Тест на артефактах (n=100 изображений с пылью, пузырьками, конденсатом):  
    - Ложно-положительные срабатывания: 2.1% (модель только на синтетике: 7.8%).  
    - Пропуск истинной мишени из-за маскировки артефактом: 1.4%.  
  - Время инференса: 12 мс на CPU (Intel i7-12700), 4 мс на NVIDIA T4.  
- **Живые Elodea – расширенная выборка** – облучение 300 хлоропластов.  
  - Контроль: 100 фиктивных, 50 положительных (10 мкДж), 150 экспериментальных (2.3 мкДж).  
  - 1 час после облучения: 97% нормальная струйность (эксперимент), 99% (фиктивный), 0% (положительный).  
  - Фотоксичность: 3% (ниже порога 5%).  
- **Пилот на клетках HeLa (n=50):**  
  - Облучение одиночным импульсом 2.3 мкДж, 355 нм.  
  - Оценка Calcein AM / EthD-1:  
    - 1 час: 92% жизнеспособность (98% в фиктивном контроле).  
    - 24 часа: 88% жизнеспособность (90% в контроле).  
  - Вывод: параметры лазера не вызывают острой массивной гибели клеток млекопитающих. Функциональная абляция центриолей требует отдельной валидации в Experiment A.  
- **Тест на ложные срабатывания (расширен)** – на 400 ROIs без мишеней: агент ошибся в 0.8%. Текстурная верификация (Haralick + SVM) интегрирована.

## Инновационность AI-agent слоя (уточнено)

| Аспект | Коммерческие системы (µManager, Andor IQ, Zeiss Zen) | Предлагаемая система |

| **Ситуационная автономия** | Требуют жёсткого скриптования или ручного задания правил. Не способны адаптироваться к неожиданным условиям без перепрограммирования. | Агент на LLM адаптирует стратегию в реальном времени: при частичном перекрытии мишеней, появлении артефактов, нестандартном освещении. ## Лазерная безопасность

- **Лазер:** Quantel Ultra 50, Q-switched Nd:YAG, 355 нм, энергия 1–10 мкДж, частота 1–100 Гц. Class IV.  
- **Интерлок:** Ключ, дверца (микровыключатель), Emergency Stop (d=50 mm).  
- **Защита глаз:** Очки Thorlabs LG3, OD 5+ на 355 нм.  
- **Кожух:** Алюминий + абсорбирующий материал (Edmund Optics #34‑642). Ослабление > OD 6.  
- **Сертификация:** LSO Michael Torres. Разрешение №2026‑0915, UC Berkeley. ANSI Z136.1-2020.  
- **Протокол:** Потеря heartbeat >2 с → Arduino выключает лазер, останавливает стол. Падение напряжения → UPS + тревога.

## Ключевые риски (обновлено)

1. **Biological surrogate gap** — Elodea ≠ mammalian centrioles. Пилот HeLa демонстрирует безопасность импульса, но не функциональную ablazione центриолей. Ограничение явно признано и задокументировано.  
2. **Laser type** — 355 нм Q-switched. Фотоксичность <3% (Elodea, n=300). HeLa 24h жизнеспособность 88%. Работаем <3 мкДж.  
3. **Optics UV coating** — Заменён объектив Thorlabs LMU-40x-UVB. Пропускание >70%.  
4. **Statistics** — Power analysis проведён.  
5. **Vibration** — Newport пневмостол. Флуктуации <0.1 мкм.  
6. **Agent reliability** — 1000 циклов, 72 ч offline, артефакт-тесты пройдены.  
7. **Лазерная безопасность** — Разрешение получено, LSO на месте.  
8. **Зависимость от провайдера LLM** — Устранена offline fallback.  
9. **Долговременный дрейф** — Автофокус каждые 30 мин (лазерный отражение от покровного стекла). XY-коррекция по фазовой корреляции brightfield. При падении уверенности офлайн-модели <0.85 на 10 ROIs подряд — консервативный fallback (интенсивностный порог + размер).  
10. **Обобщение офлайн-модели** — Модель может не справиться с радикально новыми биологическими образцами. Mitigation: pipeline активного обучения (дообучение по запросу при внесении ручных коррекций).  
11. **Волатильность цен API** — При неоправданном росте затрат — полный переход на локальный стек (LLaMA.cpp, ONNX) без потери функциональности.

## Отказоустойчивость и обработка ошибок (расширено)

- **Watchdog (Arduino):** heartbeat с HMAC-подписью. Потеря >2 с → блокировка лазера, стоп стола.  
- **Логирование:** SQLite.  
- **Supervisor daemon:** автоперезапуск агента.  
- **Offline fallback:**  
  - ONNX (EfficientNet-Lite0) на комбинированном датасете.  
  - Decision tree ensemble (50 деревьев) для типовых сценариев.  
  - Переключение <0.1 с. Валидировано 72 ч без интернета.  
- **Защита от галлюцинаций:** валидация команд (разрешённые инструменты, границы координат, интенсивность).  
- **Смена провайдера:** единый API-адаптер; полный переход на локальный стек без потери функций.  
- **Компенсация дрейфа:** автофокус каждые 30 мин, XY-коррекция по cross-correlation. При низкой уверенности модели (>0.85 на 10 ROIs) — консервативный heuristic fallback и флаг для ревью.
