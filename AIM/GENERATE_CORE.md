Дополнение к алгоритму:

Если на Level 2 (Core) нет THEORY.md / DESIGN.md / PARAMETERS.md / EVIDENCE.md — не пропускать, а создать их из пофикшенного CONCEPT.md.

Правила генерации:
- THEORY.md — научная теория/формализм из CONCEPT
- DESIGN.md — архитектура и имплементация
- PARAMETERS.md — параметры, константы, уравнения
- EVIDENCE.md — доказательства, ссылки, данные

После создания — прогнать Level 2 (Core) через TBPR с этими файлами как обычно (9 агентов).
Потом Level 3 (Full) — все файлы вместе.
