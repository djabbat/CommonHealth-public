# OPEN_PROBLEMS.md — SSA (AIM/SSA)

Актуальные нерешённые вопросы и ограничения Системного Синдромального Анализа (SSA) полного клинического анализа крови. Документ фиксирует проблемы, требующие методологического, статистического или инженерного решения для перехода от прототипа к клинически валидному инструменту.

## 1. Референсные интервалы: неполнота охвата популяций

| Проблема | Описание | Влияние | Статус |
|----------|----------|---------|--------|
| **Взрослые** | ICSH 2014 [1] задаёт границы для здоровых взрослых 18–65 лет, но не охватывает всех национальных и расовых подгрупп. | Систематическое смещение зон L0 в некоторых популяциях (например, Duffy‑null нейтропения у лиц африканского происхождения [2]). | Базовый набор ranges принят, но не верифицирован на локальных данных. |
| **Педиатрия** | Референсные интервалы для детей (<18 лет) фрагментарны; возрастная динамика (особенно первые 2 года) требует отдельных норм для каждого месяца/года [3]. | Ошибка классификации зон у детей — риск false negative по анемии и лейкопении. | Требуется отдельная локализация (планируется педиатрическая ветка SSA‑PED). |
| **Беременность** | Физиологическая гемодилюция, повышение WBC и PLT — референсы ВОЗ для Hb [4] отличаются от ICSH. | Неверное выделение анемии и воспалительных паттернов. | Доступны только рекомендации ВОЗ; единый стандарт отсутствует. |
| **Высокогорье** | Компенсаторный эритроцитоз (Hb, RBC, HCT) сдвигает все эритроидные зоны. | Ложноположительная маркировка H1/H2 по эритроцитарным параметрам. | Необходимо встраивание коррекции по высоте (например, формулы из [5]). |
| **Расовые/этнические особенности** | Помимо Duffy‑null, существуют различия по MCV, MCH, лейкоцитарным показателям [6]. | Снижение специфичности паттернов (особенно ANEMIA_MICROCYTIC_SET — ЖДА vs талассемия). | Пока не учтено; конфигурация ranges по регионам — в backlog. |

## 2. Полнота синдромальных паттернов

- **Парные комбинации:** 15 параметрических пар × 25 ячеек (5×5 зон) = 375 возможных комбинаций. Из них клинически осмысленными признаны ~80–120 (оценка экспертов). Остальные — шум или редкие девиации, требующие медицинской интерпретации. *Риск: неполное покрытие редких, но важных сочетаний (например, HGB↑ + MCV↓ при некоторых гемоглобинопатиях).*
- **Тройные (синдромальные) паттерны:** текущий список из 15 синдромов — экспертный и заведомо неполный. Отсутствуют, например:
  - **Hemoglobinopathy pattern** (HGB↓ + MCV↓ + RBC↑ + RDW→ + RET→) — β‑талассемия minor.
  - **Iron refractory iron deficiency anemia (IRIDA)** — HGB↓ + MCV↓ + RDW→ + RET→ + ферритин↑ (требует биохимии, но CBC‑паттерн отличим от ЖДА).
  - **Myelophthisic pattern** (лейкоэритробластическая картина без явного фиброза).
- **Масштабируемость:** комбинаторное пространство 5²⁸ ≈ 7.45×10¹⁹; покрытие всеми паттернами невозможно. Необходима стратегия автоматического выявления новых паттернов (data‑driven clustering на больших выборках CBC).

## 3. 5-зонная гипотеза: границы клинического риска

Пятизонная шкала (L2, L1, L0, H1, H2) использует два типа порогов:
- **Статистические (population):** 2.5‑й / 97.5‑й перцентили (ICSH).
- **Клинические критические (clinical critical):** значения, при которых требуется немедленное вмешательство (например, Hb < 60 г/л, PLT < 20×10⁹/л, WBC < 1.0×10⁹/л).

**Проблема:** для многих параметров (RDW, MPV, PDW, PCT, лимфоциты %) отсутствуют общепринятые клинические критические пороги. ICSH даёт рекомендации, но не всегда числовые значения. *Предлагаемое решение:* взять пороги из BCSH‑guidelines [7], для остальных — экспертный консенсус с последующей валидацией.

| Параметр | ICSH‑статистический (L0) | Клинический критический (L2/H2) | Источник |
|----------|--------------------------|----------------------------------|----------|
| HGB | 120–160 г/л (♀), 130–175 г/л (♂) | <60 г/л ; >200 г/л (криз) | BCSH [7] |
| PLT | 150–400×10⁹/л | <20×10⁹/л ; >1000×10⁹/л | BCSH |
| NEUT_abs | 1.5–7.0×10⁹/л | <0.5×10⁹/л (агранулоцитоз) | BCSH |
| LYMPH_abs | 1.0–4.0×10⁹/л | <0.5×10⁹/л ; >10.0×10⁹/л | Экспертно |
| RDW | 11.5–14.5% | >25% (тяжёлая анизоцитоз) | Экспертно (нет BCSH) |
| MPV | 7.2–11.7 fL | <6.0 fL ; >13.0 fL | Экспертно |

## 4. Лабораторные артефакты и преаналитические искажения

Артефакты способны симулировать любой паттерн — критично для автоматического анализа.

| Артефакт | Искажаемые параметры | Паттерн‑мимикрия | Меры |
|----------|----------------------|------------------|------|
| EDTA‑зависимая псевдотромбоцитопения | PLT↓↓ | THROMBOTIC_MICROANG, ITP | Флаг при PLT↓ + MPV→ + отсутствии клиники; запрос цитратной пробы |
| Холодовые агглютинины | RBC↓, HCT↓↓, MCV↑↑, MCHC↑ | MACROCYTIC_ANEMIA, гемолиз | Флаг при MCV↑ + MCHC↑ + RBC↓ + отсутствии ретикулоцитоза; предварительный подогрев |
| Гипертриглицеридемия (липемия) | HGB↑ (спектрофотометрически), MCHC↑ | MYELOPROLIFERATIVE (ложный HGB) | Флаг при MCHC > 370 г/л без клиники; проверка плазмы на липемию |
| Лейкоцитоз > 100×10⁹/л | HGB↓ (мутность), HCT↑ (объём лейкоцитов) | ANEMIA + макроцитоз | Алгоритм коррекции: corrected HGB = HGB_measured – (WBC×0.0005) [8] |
| Длительное хранение пробы (>4 ч) | MCV↑, RDW↑, PLT↓, MPV↑ | MACROCYTIC, анизоцитоз, тромбоцитопения | Предупреждение при времени >4 ч в метаданных |

**Текущее состояние:** в API введён флаг `preanalytical_flags` (опционально), но алгоритмы mitigation не реализованы.

## 5. Калибровка LLM‑ранжировщика

После первичного матчинга по таблицам паттернов выполняется LLM‑реранжировка (через `llm.py`), которая должна уточнить ранжированный список синдромов. **Проблема:** LLM не откалибрована — вероятности (confidence scores) не соответствуют реальной частоте правильности. *Последствия:* при top‑1 accuracy 70% калибровочная ошибка (ECE) может превышать 0.2, что делает вывод ненадёжным для клинического использования.

**План исправления:**
1. Собрать gold‑set из 200+ CBC с верифицированными синдромами.
2. Получить от LLM confidence для каждого паттерна.
3. Применить калибровку Платта или изотоническую регрессию.
4. Повторно оценить ECE и Brier score.

## 6. Multi‑label синдромы: один CBC — несколько паттернов

Текущая архитектура возвращает **ранжированный список**, подразумевая, что CBC соответствует единственному синдрому. На практике:
- Тромботическая микроангиопатия: одновременно PANCYTOPENIA (PLT↓↓, HGB↓) + HEMOLYSIS_PATTERN (RETIC↑, шистоциты) + THROMBOTIC_MICROANG.
- Сепсис с ДВС: SEPSIS_DECOMP + THROMBOTIC_MICROANG.
- Миелодиспластический синдром: может включать MACROCYTIC_ANEMIA + BICYTOPENIA + признаки дисплазии.

**Необходимо:** перейти от однозначного ранжирования к multi‑label classification, где выводится набор синдромов (возможно, с перекрёстными вероятностями). *Проблема:* отсутствие gold‑set с multi‑label разметкой — требует отдельного сбора данных.

## 7. Контракт SSA → DiffDiagnosis

SSA предполагается как вход для подпроекта DiffDiagnosis, который по синдромам генерирует нозологический дифференциальный ряд. На данный момент контракт не формализован:

- **Формат передачи:** ранжированный список синдромов (JSON), но без указания силы связи (уверенность, likelihood).
- **Пороговые значения:** какие синдромы передавать (top‑3, top‑5, все с вероятностью >0.1)?
- **Контекстные переменные:** необходима ли передача исходного CBC (для пересчёта индексов внутри DiffDiagnosis)?
- **Иерархия:** некоторые синдромы являются подмножествами других (например, PANCYTOPENIA включает BICYTOPENIA). Как избежать дублирования?

*Предварительное решение:* использовать все синдромы с вероятностью >0.15, передавать как массив объектов `{syndrome_id: string, confidence: float, level: “pattern” | “subpattern”}`, и сохранять биекцию к CBC.

## 8. Педиатрическая таксономия (SSA‑PED)

Дети до 2 лет имеют:
- Физиологический лимфоцитоз (LYMPH_abs до 10×10⁹/л).
- Более низкие нормы Hb (возрастная анемия недоношенных).
- Другие пороги для PLT (норма 150–450, но у новорождённых до 600).
- Отсутствие зрелых нейтрофилов (бэнды в норме).

Создание отдельной ветки SSA‑PED требует:
1. Формализации возрастных групп (0–28d, 1–12m, 1–2y, 2–6y, 6–12y, 12–18y) с уникальными ranges.
2. Переопределения всех 15 парных и 15 тройных паттернов для каждой группы (например, VIRAL_PATTERN у детей — нередко норма, а не патология).
3. Валидации на педиатрических наборах данных.

## 9. Регуляторные требования (MDR / SaMD)

Подпроект SSA — составная часть клинического инструмента поддержки принятия решений (CDSS). В перспективе он может подпадать под классификацию медицинского программного обеспечения (SaMD) по классификации IMDRF или EU MDR (класс IIa и выше). На R&D стадии это не критично, но архитектурные решения должны учитывать:

- **Прослеживаемость паттернов:** каждое заключение должно быть объяснимым (какие зоны и параметры привели к синдрому).
- **Управление версиями ranges:** референсные интервалы и паттерны — версионированный артефакт (SemVer).
- **Аудит:** лог всех вычислений SSA с возможностью воспроизведения.

*Проблема:* текущий код не содержит механизмов прослеживаемости и версионирования ranges (ranges хранятся в JSON без истории изменений).

## 10. Валидационный набор (gold‑set)

**Текущий статус:** 0 размеченных примеров.
**План:** 200 CBC с верифицированными синдромами (золотой стандарт по биопсии костного мозга, молекулярной диагностике, клиническому окончательному диагнозу).

**Неразрешённые вопросы:**
- Рекрутинг: какие источники (архивы лабораторий, клинические исследования)?
- Разметка: кто и как определяет синдром для каждого CBC? (Экспертная панель из 3 гематологов, консенсус).
- Классовый дисбаланс: распространённые синдромы (INFLAMMATION, ANEMIA_MICROCYTIC_SET) будут преобладать; редкие (EOS_HYPERSYNDROME, LEUKOERYTHROBLASTIC) — единичны. Как оценивать производительность на редких синдромах?
- Multi‑label разметка: требуется ли для gold‑set указание всех присутствующих синдромов (а не только основного)?

*Текущее решение:* разметка только основного синдрома (single‑label) для начальной калибровки; multi‑label — вторая фаза.
