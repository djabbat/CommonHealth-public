## 2. Импедансный блок — ODE-модель (§2–§5, §12)

Замкнутая скалярная модель импеданса агента:

```
dI/dτ = σ(τ) − λ · I · (1 − I/I_max)
```

- `σ(τ)` — экзогенный поток новизны (sensory drive);
- `λ` — скорость обучения (снижение ошибки при не-насыщенной модели);
- `I_max` — предельный импеданс до «чистого режима обучения» (§18.3).

Производные величины:

```
t_phys(τ) = ∫₀^τ I(τ') dτ'              # физическое время
𝒞(τ)       = −dI/dτ                       # сознание
Φ_Ze(T)   = ∫₀^T I(τ) dτ                  # интегральная мера (конечная форма)
K(τ)      = −I(τ)                         # знание
```

**Сценарии:**
- **Routine:** малое σ, λ высокий → `I → 0`, `𝒞 → 0`, `t_phys` замедлен.
- **Novelty:** σ ступенькой вверх → `I` растёт, `𝒞` отрицателен (ошибка накапливается), затем позитивен (обучение), `t_phys` ускоряется.
- **Meditation:** σ → 0 и λ повышен → `I` убывает монотонно, `𝒞 > 0` константой → Φ_Ze сходится сверху.
- **Cheating spike:** `I` скачком вниз (изменение реальности под модель), не через σ.

## 3. Квантовый блок — Ze-деформация CHSH (§7, §19)

Входы: угол Алисы `a`, угол Боба `b` (в единичных векторах на Блоховой сфере, параметризуем углами).

```
E_QM(a, b) = −cos(a − b)                  # стандарт
δ(∇𝓘, H) = δ₀ · (1 − 2α·H)                # Ze-сдвиг
E_Ze(a, b) = E_QM(a, b) + δ · [cos²(a − b) − 1/3]
```

Два режима измерения:

### 3.1. Singlet-оптимальные углы

`a₁=0, a₂=π/2, b₁=π/4, b₂=3π/4`, CHSH-форма `S = |E₁₁ − E₁₂ + E₂₁ + E₂₂|`.

```
|S_QM| = 2√2 ≈ 2.8284
|S_Ze| = 2√2 − δ/3                        # все cos²(a−b) = 1/2
|ΔS|   = δ/3
```

Даёт максимальное S_QM, но малый Ze-сдвиг.

### 3.2. Shift-оптимальные углы (5.md §7.4)

`(0°, 45°, 22.5°, −22.5°)` = `(0, π/4, π/8, −π/8)`, CHSH-форма `S = |E₁₁ + E₁₂ + E₂₁ − E₂₂|`.

```
|S_QM|(shift-opt) ≈ 2.389                 # < 2√2
|ΔS|              ≈ δ · 1.7478             # максимизирует Ze-сдвиг
```

### 3.3. Квантовое демпфирование (5.md §8.4)

```
S(H) = |S_QM| · (1 − 2α·H)                # эмпирический закон
```

`α ≈ 0.03` (BBO); `H` — энтропия модулятора [0, 1].

**Предсказание (shift-оптимальные углы, δ₀ = 0.05, α = 0.03, H = 0.5):**
`|ΔS| ≈ 0.05·0.97·1.7478 ≈ 0.0848` → ~42σ при 10⁹ совпадений.

## 4. Пространственный блок — cheating autowaves (§17)

Реакционно-диффузионная система на 1D/2D-решётке. Переменные: `I` (импеданс), `x` (уровень обучения), `y` (уровень жульничества).

```
∂I/∂t = D · ∇²I + α_aw · (1 − x) · I − β_aw · x · y
∂x/∂t = γ_aw · I · (1 − x) − δ_aw · x
∂y/∂t = ε_aw · [I > I_crit] − ζ_aw · y
```

- `[P]` — индикатор (1 если P истинно, иначе 0). Сглаживается сигмоидой `1/(1 + exp(−k·(I − I_crit)))` для дифференцируемости и стабильности интегратора.
- **Фазы:**
  - low-I, high-x: стабильное обучение.
  - high-I, high-y: стабильное жульничество.
  - промежуток: автоволны (осцилляции); в 2D — спиральные волны.

Численная схема: явный Эйлер для `x, y`, ADI для `I` на 2D (в MVP — явный Эйлер на 1D-решётке с периодическими границами).

## 5. Релятивистский блок (§9) — **не в симуляторе**

Поле `𝓘(x^μ)` с действием:

```
S = ∫ d⁴x √(−g) [ R/(16πG) + ½ ∇_μ𝓘 ∇^μ𝓘 + V(𝓘) + L_mat ]
```

Уравнения поля:

```
R_μν − ½ R g_μν = 8π G (T^mat_μν + T^Ze_μν)
□𝓘 − V'(𝓘) = R / (2 Λ_Ze)
```

Космологическое решение:

```
𝓘̈ + 3H 𝓘̇ + m_Ze² 𝓘 = 3 (ä/a) / Λ_Ze
```

→ в текущем релизе не моделируется (требует GR-интегратора). Фиксировано для будущих версий.
